Ripple Color Classes

These are the new CSS classes added to the design system for button ripple effects:

.btn-ripple-primary
rgba(255, 255, 255, 0.8)
.btn-ripple-secondary
rgba(255, 255, 255, 0.6)
.btn-ripple-danger
rgba(255, 255, 255, 0.8)
.btn-ripple-success
rgba(255, 255, 255, 0.8)
.btn-ripple-warning
rgba(0, 0, 0, 0.6)
.btn-ripple-info
rgba(255, 255, 255, 0.8)
