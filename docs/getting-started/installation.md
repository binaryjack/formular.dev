# Installation

Getting started with `formular.dev` is quick and straightforward. The package is published on npm and is distributed with TypeScript definitions built-in.

## Package Manager

Install the package using your preferred package manager:

::: code-group

```bash [pnpm]
pnpm add formular.dev
```

```bash [npm]
npm install formular.dev
```

```bash [yarn]
yarn add formular.dev
```

```bash [bun]
bun add formular.dev
```

:::

## Requirements

- **Node.js**: `v18.0.0` or higher (for SSR environments)
- **TypeScript**: `v4.9.0` or higher (if using TypeScript, though JavaScript is fully supported)
- **ES Modules**: `formular.dev` is distributed as an ESM package. Ensure your environment or bundler supports ES modules.
